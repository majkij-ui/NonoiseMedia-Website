# Project Context: Nonoise Media Portfolio

## The App
A premium, high-converting portfolio website for "Nonoise Media", a high-end video production and cinematic storytelling studio based in Warsaw, Poland. The aesthetic is highly editorial, minimalist, and "magazine-like" (dark mode, strict typography, narrow text columns, massive image grids). The vibe is "show, don't tell"—quiet confidence, rigorous craftsmanship, and uncompromising quality.

## The Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (heavy use of scroll-triggered `whileInView`, staggered reveals, and `layoutId` shared layout animations)
- **Hosting:** Vercel
- **Media CDN:** Cloudflare R2 (Custom domain: `https://nonoise.media`)

## Target User
Premium brands, agencies, and corporate clients (e.g., PHH, Danone, CIONET) looking for high-end commercial, event, and documentary video production.
